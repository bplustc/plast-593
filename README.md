# Plast593

Sitio web para la empresa Plast593: fabricación y venta de productos plásticos de alta calidad en Ecuador.

## Dominio de producción
https://plast593.com/

## Características principales
- Responsive: optimizado para escritorio y móvil.
- Catálogo de productos y servicios.
- Modal de carrito y formulario de pedido.
- Accesibilidad y navegación clara.
- SEO optimizado (meta etiquetas, Open Graph, canonical, textos alternativos, estructura semántica).

## Estructura del proyecto
- index.html: Página principal
- productos.html: Catálogo de productos
- servicios.html: Servicios ofrecidos
- acerca.html: Información de la empresa
- contacto.html: Formulario de contacto
- styles.css: Estilos globales y responsive
- scripts.js: Lógica de interacción y renderizado
- images/: Imágenes y logotipos

## Deploy
1. Sube todos los archivos y carpetas al hosting o servidor.
2. Asegúrate de que el dominio apunte a la carpeta raíz del proyecto.
3. Verifica que las rutas de imágenes y scripts sean correctas.
4. Revisa el sitio en dispositivos móviles y escritorio.

## SEO y Accesibilidad
- Meta etiquetas y Open Graph configurados.
- Imágenes con atributos alt descriptivos.
- Navegación accesible con roles y aria-label.
- Estructura semántica HTML5.
