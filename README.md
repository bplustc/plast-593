# Metal593

Sitio web para Metal593: Diseño, Fabricación y Puesta en Marcha de Proyectos Mecánicos Industriales.

## Características
- Responsive, moderno y optimizado para móviles
- Galería visual destacada
- Navegación accesible y rápida
- SEO básico implementado
- Código HTML5 y CSS3 limpio

## Estructura
- `index.html`: Página principal
- `nuestra-empresa.html`: Información de la empresa
- `proyectos-servicios.html`: Proyectos y servicios ofrecidos
- `contacto.html`: Redes sociales y contacto
- `styles.css`: Estilos globales
- `script.js`: Lógica de menú hamburguesa
- `images/`: Imágenes y logotipos

## SEO y Producción
- Etiquetas meta para SEO y redes sociales
- Títulos y descripciones optimizadas
- Uso de fuente Montserrat para mejor legibilidad
- Imágenes optimizadas y con atributos alt descriptivos
- Navegación accesible y enlaces claros

## Despliegue
Sube todos los archivos a tu hosting o Netlify/Vercel. El sitio es estático y no requiere backend.
